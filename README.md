  ## 🖥️ Projeto 
  
  O DevLinks é um agregador de links para usar como cartão de visitas online.

  ## 🚀 Tecnologias
  Esse projeto foi desenvolvido durante o discover da Rocketset, com as seguintes tecnologias:

  - HTML
  - CSS
  - Javascript
  - Git e Github
